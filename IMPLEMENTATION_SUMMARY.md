## 7. **New Database Models** ✓

### Model 1: `UserCareerGoal`
- Tracks user's target career
- Stores timeline and proficiency targets
- One-to-one relationship with User

### Model 2: `LearningGoal`
- Tracks learning goals for specific skills
- Supports deadline and proficiency targets
- Unique constraint (user, skill)

### Model 3: `LearningProgress`
- Logs hours spent learning
- Tracks completion percentage
- Stores milestone achievements
- Stores completed courses list

### Model 4: `CareerPivotAnalysis`
- Stores AI analysis of career transitions
- Includes readiness score and recommendations
- Can be generated by AI or manually
